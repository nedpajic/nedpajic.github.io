# Ned Pajic Portfolio

A static portfolio website built with HTML, CSS, and a small amount of JavaScript.

## Preview locally

Double-click `index.html` or open it in any browser.

## Publish with GitHub Pages

1. Create a public repository named `nedpajic.github.io`.
2. Upload all files and folders from this package to the repository root.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**, select `main`, and use `/ (root)`.
5. Visit `https://nedpajic.github.io` after GitHub finishes publishing.

## Before publishing

Replace any placeholder or outdated contact links as needed. The included resume and portfolio PDFs live in the `assets` folder.
